# Changelog

## [1.0.0] — 2026-05-08

### Added
- Initial public release
- JSON-to-Graph engine for instant architecture visualization
- Interactive Cytoscape.js canvas (pan, zoom, box-select, drag)
- Dynamic Edge Routing: Smart orthogonal (Taxi) and smooth (Bezier) lines
- Directional auto-correction to prevent backward-piercing arrows
- Dagre auto-layout engine for instant hierarchy mapping
- Custom Node Painter with 24-color SaaS palette + hex picker
- Undo/Redo history stack (Ctrl+Z / Ctrl+Y)
- 24px Snap-to-Grid engine for perfect linear alignment
- Dark / Light theme toggle
- Export to high-resolution PNG (Scale: 6x)
- Export to JSON (persists node coordinates)
- `localStorage` persistence for layout and preferences
- Zero-build-step deployment (Single HTML file)
