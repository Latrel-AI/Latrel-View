Syntax Guide & LLM Prompt

1. LLM Generation Prompt

Copy/paste this into ChatGPT or Claude to instantly generate architectures:

You are an expert cloud architect. Create a JSON array representing the architecture diagram for [INSERT YOUR USE CASE HERE]. 

Strict Rules:
- Output ONLY a valid JSON array. No markdown, no explanations.
- Nodes must have: "data": {"id": "unique_id", "label": "Short Name", "parent": "group_id" (optional), "type": "db" (optional for database shape)}.
- Edges must have: "data": {"id": "e1", "source": "node1", "target": "node2", "label": "Action"}.
- Group logical phases/tiers using parent nodes.

2. Node Syntax

{ "data": { "id": "api", "label": "API Gateway", "color": "#0f172a" } }
  • id — Unique identifier (required)
  • label — Display text
  • parent — ID of a parent node (for nested grouping)
  • type — Set "db" for database cylinder shape

3. Edge Syntax

{ "data": { "id": "e1", "source": "api", "target": "db", "label": "query" } }

4. Keyboard Shortcuts

  • Ctrl+drag on canvas — Box-select multiple nodes
  • Ctrl+click a node — Add/remove from selection
  • Ctrl+Z / Ctrl+Y — Undo / Redo
